# Contributing

Thanks for contributing to HeartSync!

## Development

```bash
npm install
node --check bin/cli.js
npm pack --dry-run
```

## Pull Requests

- Keep PRs focused and small.
- Update docs when behavior changes.
- Do not include secrets, tokens, or private keys.
- Keep naming and paths consistent with current branding.

## Commit Style

Use clear prefixes when possible:
- `feat:` new functionality
- `fix:` bug fixes
- `docs:` documentation only
- `chore:` maintenance

## Security

If your finding is security-related, please follow `SECURITY.md` instead of opening a public issue.
