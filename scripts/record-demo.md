# Demo Video Recording (30-45s)

## 1) Start OpenClaw + skill
```bash
openclaw gateway start
```

## 2) Open your chat surface
Use Telegram/Discord where your agent is connected.

## 3) Record script (say these exact lines)
1. "Send me a selfie"
2. "Show me your date-night look"
3. "Good night selfie"

## 4) Export
- Filename: `assets/demo.mp4`
- Recommended: 1280x720, < 15MB

## 5) Commit
```bash
git add assets/demo.mp4 README.md scripts/record-demo.md
git commit -m "docs: add demo video"
git push
```
